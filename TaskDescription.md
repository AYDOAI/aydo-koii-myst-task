# AYDO Koii Myst Task

The AYDO Koii Myst Task integrates Mysterium Network's decentralized VPN infrastructure with Koii's attention tracking system. This task allows users to deploy and manage Mysterium nodes, ensuring network reliability and performance. Participants earn KPL tokens for maintaining node uptime, bandwidth contribution, and network stability. The task leverages Koii's consensus mechanism to verify node performance and distribute rewards, creating a sustainable ecosystem for decentralized privacy services.
