# AYDO Koii Myst Task

The AYDO Koii Myst Task integrates Mysterium Network's decentralized VPN infrastructure with Koii's attention tracking system. This task allows users to deploy and manage Mysterium nodes, ensuring network reliability and performance. Participants earn KOII & MYST tokens for maintaining node uptime, bandwidth contribution, and network stability. The task leverages Koii's consensus mechanism to verify node performance and distribute rewards, creating a sustainable ecosystem for decentralized privacy services.

## Configuration

To receive rewards, you must configure the following environment variable:

* **`BENEFICIARY_WALLET`**
  **Description:** Your earnings will automatically be paid out to the wallet address submitted below. If you have not set your wallet or would like to update it, please do it now. Please make sure the address is an **ERC-20 Polygon-compatible wallet**, e.g. MetaMask.

Make sure to correctly set this variable before deploying your node to ensure proper reward distribution.
